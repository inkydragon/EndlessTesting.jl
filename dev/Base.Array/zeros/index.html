Float64

julia> zeros()

0.0

Vector

julia> zeros(1)

 0.0

julia> zeros(2)

 0.0
 0.0

Array

julia> zeros(1, 1)

 0.0

julia> zeros(1, 2)

 0.0  0.0

julia> zeros(2, 1)

 0.0
 0.0

julia> zeros(2, 2)

 0.0
